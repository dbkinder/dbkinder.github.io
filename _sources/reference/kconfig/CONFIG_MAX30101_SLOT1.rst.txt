.. title:: MAX30101_SLOT1

.. option:: CONFIG_MAX30101_SLOT1
.. _CONFIG_MAX30101_SLOT1:

Set which LED and pulse amplitude are active in time slot 1.
0: None (disabled)
1: LED1 (red), LED1_PA
2: LED2 (IR), LED2_PA
3: LED3 (green), LED3_PA
4: None (disabled)
5: LED1 (red), PILOT_PA
6: LED2 (IR), PILOT_PA
7: LED3 (green), PILOT_PA
